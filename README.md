# 🎥 AI-Powered YouTube Transcriber & Summarizer

✨ Turn long YouTube videos into quick, meaningful summaries in seconds.

---

## 🚀 Overview

This project is an intelligent AI-based system that:
- 🎧 Extracts audio from YouTube videos
- 🧠 Transcribes speech using Whisper
- ✍️ Generates concise summaries using Large Language Models (LLMs)

It helps users quickly understand long video content without watching the entire video.

---

## 🧠 Features

🔍 Automatic YouTube video processing  
🎧 Audio extraction using pytube  
📝 Speech-to-text transcription using Whisper  
🤖 AI-based summarization using LLM  
⚡ Fast and efficient pipeline using Haystack  

---

## 🏗️ Workflow

➡️ Input YouTube URL  
➡️ Download video audio  
➡️ Transcribe using Whisper  
➡️ Process text using LLM  
➡️ 📊 Generate summary
